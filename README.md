# 🎬 MatchTube

MatchTube is a social browser extension that connects your YouTube watch history with your friends. It allows you to see who else in your group has watched the same videos, turning a solitary viewing experience into a shared one.

## 🚀 How it works

1.  **Automatic Logging**: Whenever you watch a YouTube video, MatchTube securely logs the video ID and your username to your private Supabase database.
2.  **Real-time Matching**: When you open a YouTube video, the extension checks if any of your friends have watched the same video in the last 7 days.
3.  **Group Sync**: You can view a consolidated list of all videos that have been watched by more than one person in your group.

> [!TIP]
> To share with friends, simply have everyone use the **same Supabase URL and API Key** in their settings, but choose unique usernames!

---

## 🛠 Database Setup (Supabase)

To use MatchTube, you need a Supabase project to act as the central hub for your group's watch history.

### 1. Create a Supabase Project
Go to [Supabase](https://supabase.com/) and create a new project. (It's free to have up to 2 active projects running :))

### 2. Create the Table
Navigate to the **SQL Editor** in your Supabase dashboard and run the following script to create the `watch_history` table:

```sql
-- Create the watch history table
CREATE TABLE watch_history (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  created_at TIMESTAMPTZ DEFAULT NOW() NOT NULL,
  video_id TEXT NOT NULL,
  username TEXT NOT NULL,
  video_title TEXT,
  
  -- Prevent duplicate entries for the same user and video
  UNIQUE(video_id, username)
);

-- Enable Row Level Security (RLS)
ALTER TABLE watch_history ENABLE ROW LEVEL SECURITY;

-- Create a policy that allows anyone with the API key to insert and select
-- (For a simple setup with friends)
CREATE POLICY "Allow public access" ON watch_history
FOR ALL USING (true) WITH CHECK (true);
```

### 3. Get your API Keys
1.  Go to **Project Settings** > **API**.
2.  Copy your **Project URL**.
3.  Copy your **anon (public) API Key**.

## 📦 Installation

1.  Clone this repository or download the source code.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (toggle in the top right).
4.  Click **Load unpacked** and select the folder containing this extension.

---

## ⚙️ Configuration

1.  Click the **MatchTube icon** (🎬) in your browser toolbar.
2.  Go to the **Settings** tab (the gear icon ⚙️).
3.  Enter your **Username** (how you will appear to friends).
4.  Paste your **Supabase URL** and **Anon Key**.
5.  Click **Save Configuration**.

Now, whenever you watch a YouTube video, you'll be able to see if your friends has seen it too!

---

## 🔒 Privacy Note
This extension only tracks the `video_id` and `video_title` of YouTube videos you watch. It does not access your personal YouTube account, email, or any other browsing data. Since you host your own database on Supabase, you have full control over your data.
